# CSE 155 Class Project

Base repo for our webcam-based gesture game.

## Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`

## Stack
- Vite
- TypeScript
- Phaser

## Branch workflow
- Please DO NOT work off main. Create your own branch and push all your work to the branch.
- Use pull requests to push to `main`.
